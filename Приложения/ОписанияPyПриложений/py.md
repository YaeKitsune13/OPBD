## 1. Скрипт first.py (Связанные таблицы)

**Библиотеки:** `PyQt6`, `PyQt6.QtSql`.

**Параметры подключения:**

```python
MYSQL_HOST, MYSQL_DB, MYSQL_USER, MYSQL_PASS = "127.0.0.1", "insurance", "root", "1234"
PG_HOST, PG_DB, PG_USER, PG_PASS = "127.0.0.1", "insurance", "postgres", "1234"
```

**Микро-вывод:** Код содержит настройки для двух разных баз.

**Установка соединения:**

```python
db_ms = QSqlDatabase.addDatabase("QMARIADB", "mysql_conn")
db_pg = QSqlDatabase.addDatabase("QPSQL", "pg_conn")
# ... установка параметров и открытие .open()
```

**Микро-вывод:** Программа подключается к MySQL и PostgreSQL одновременно.

**Вывод данных:**
Используется `QTableView` и `QSqlQueryModel` для отображения списка сотрудников и их страховых случаев.

```python
self.master_model.setQuery(SQL_EMPLOYEES)
self.master_view.setModel(self.master_model)
```

**Микро-вывод:** Скрипт выводит данные в классические таблицы.

## 3. Скрипт third.py (Двойной фильтр данных)

**Библиотеки:** Стандартный пакет `PyQt6.QtSql`.

**Параметры и соединение:**
Стандартное подключение к базе `insurance`.
**Микро-вывод:** Программа стабильно работает с существующей базой данных.

**Вывод данных:**
Реализована цепочка: Выбор сотрудника -> Выбор его клиента -> Таблица случаев.

```python
query.prepare("SELECT policy_number, full_name FROM policyholders WHERE employee_id = :emp_id")
query.bindValue(":emp_id", emp_id)
query.exec()
```

**Микро-вывод:** Двойная фильтрация позволяет быстро найти нужный полис.

## 4. Скрипт fourth.py (Вывод в виде карточек)

**Библиотеки:** `PyQt6` (виджеты `QFrame` и `QScrollArea`).

**Параметры и соединение:**
Используются те же данные для MySQL и Postgres.
**Микро-вывод:** Подключение к СУБД происходит в фоновом режиме.

**Вывод данных:**
Данные выводятся не в таблицу, а в красивые графические карточки через отдельный класс `ClaimCard`.

```python
while query.next():
    card = ClaimCard(client_name=query.value("client_name"), ...)
    self.cards_layout.addWidget(card)
```

**Микро-вывод:** Карточный интерфейс выглядит современно и очень наглядно.
