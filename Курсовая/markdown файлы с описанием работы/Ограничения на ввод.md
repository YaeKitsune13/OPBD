# Ограничения на ввод данных

## 1. Профиль и Авторизация
| Поле | Ограничение | Сообщение (Toast / Hint) |
| :--- | :--- | :--- |
| **Email** | Наличие @ и домена | «Введите почту и пароль» (при пустом) |
| **Пароль** | Минимум 8 символов | «Минимум 8 символов» (placeholder) |
| **Обязательные поля** | Не пустые Фамилия/Имя | «Заполните обязательные поля» |
| **Роли** | Только из справочника | Автоматическое назначение при входе |

## 2. Картотека питомцев
| Поле | Ограничение | Подсказка / Логика в UI |
| :--- | :--- | :--- |
| **Кличка** | Только буквы, 1-30 симв. | «От 1 до 30 символов, только буквы» |
| **Вес (кг)** | Число от 0.1 до 300.0 | Атрибуты min="0.1" max="300" step="0.1" |
| **Дата рождения** | Не может быть в будущем | Тип input="date", валидация при сохранении |
| **Фото** | Форматы JPG, PNG | Атрибут accept=".jpg,.png" |
| **Размер фото** | Не более 5 МБ | Ограничение серверной части (указано в UI) |

## 3. Ведение приёма (Интерфейс врача)
| Поле | Ограничение | Логика валидации |
| :--- | :--- | :--- |
| **Анамнез** | От 10 до 2000 символов | Обязательное поле (пометка красной звёздочкой) |
| **Диагноз** | Текстовое поле | Обязательное поле для сохранения «Медкарты» |
| **Вес на приёме** | Число от 0.1 до 300.0 | Автоматически обновляет профиль питомца |
| **Назначения** | Минимум 1 позиция | Услуга или Медикамент (выбор из справочника) |
| **Удаление** | Подтверждение | Сообщение «Удалено» (Toast-error) |

## 4. Запись на приём (Интерфейс клиента)
| Поле | Ограничение | Логика UI |
| :--- | :--- | :--- |
| **Дата приёма** | `min` = завтрашний день | Атрибут min="2026-04-29" (в календаре) |
| **Специализация** | Обязательный выбор | Пока не выбрана, список врачей заблокирован |
| **Врач** | Фильтрация | Отображаются только врачи выбранного профиля |
| **Время** | Свободные слоты | Выбор из выпадающего списка (шаг 30 мин) |

## 5. Управление справочниками (Администратор)
| Поле | Ограничение | Логика UI |
| :--- | :--- | :--- |
| **Цена** | Число > 0 | Отрицательные значения заблокированы (min="1") |
| **Срок годности** | Формат ММ.ГГГГ | Подсветка красным в таблице при истечении срока |
