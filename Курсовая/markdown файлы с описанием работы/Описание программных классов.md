# 2.1 Описание программы

Программный продукт разработан на стеке **Vue 3 + Vite** (frontend) с REST API на backend. Все модули разделены на UI-компоненты и контроллеры API.

## UI-компоненты

| Компонент | Props (Вход) | Emits (События) | Назначение |
| :--- | :--- | :--- | :--- |
| `TopBar.vue` | `userName`, `userAvatar` | `logout`, `open-profile` | Шапка сайта — отображает логотип, имя пользователя, кнопку выхода |
| `BaseModal.vue` | `show`, `title`, `maxWidth` | `close` | Универсальное модальное окно для форм и подтверждений |
| `SideBar.vue` | `currentRole`, `activePage` | `navigate` | Боковое меню навигации — набор пунктов зависит от роли пользователя |
| `PetCombobox.vue` | `pets` | `select` | Кастомный выпадающий список питомцев с поиском и закрытием по клику вне |
| `PrintZone.vue` | `visitData` | — | Область печатной формы, скрытая до вызова `window.print()` |
| `ProfilePanel.vue` | `userData` | `close`, `save` | Боковая панель редактирования профиля пользователя |

## Backend API (Контроллеры)

| Класс | Методы | Тип | Назначение |
| :--- | :--- | :---: | :--- |
| **AuthController** | `login`, `register`, `logout` | POST | Авторизация и регистрация пользователей, определение роли по домену email |
| **PetController** | `getByOwner`, `addPet`, `updatePet`, `delete` | GET/POST/PUT/DELETE | Управление картотекой питомцев владельца |
| **VisitController** | `saveVisitCard`, `getJournal`, `getById` | POST/GET | Создание и получение медицинских карт визитов |
| **AppointmentController** | `create`, `getByOwner`, `updateStatus`, `cancel` | POST/GET/PUT | Управление заявками на приём, изменение статусов |
| **DoctorController** | `getSchedule`, `getBySpecialty` | GET | Получение расписания врача, фильтрация по специализации |
| **AdminController** | `getStats`, `getRevenue`, `manageServices`, `manageMeds` | GET/PUT/DELETE | Аналитика, финансовые отчёты, управление справочниками |
| **WeightController** | `getHistory`, `addRecord` | GET/POST | Получение и добавление записей динамики веса питомца |
