# Описание входной и выходной информации

## Входная информация

### Профиль и Регистрация (форма Вход/Регистрация)
| Поле | Источник | Формат | Обязательность |
| :--- | :--- | :--- | :--- |
| Фамилия | input: regLastName | Текст | Да |
| Имя | input: regFirstName | Текст | Да |
| Отчество | input: regMiddleName | Текст | Нет |
| Телефон | input: regPhone | +7XXXXXXXXXX | Да |
| Email | input: regEmail | email | Да |
| Адрес | input: regAddress | Текст | Нет |
| Пароль | input: regPass | Строка (мин. 8 симв.) | Да |

### Карточка питомца (модальное окно Добавить/Изменить)
| Поле | Источник | Формат | Примечание |
| :--- | :--- | :--- | :--- |
| Кличка | input: text | Текст (только буквы) | 1-30 симв. |
| Вид животного | select | Список (Кошка, Собака, Кролик...) | Выбор из 6 поз. |
| Порода | input: text | Текст | — |
| Дата рождения | input: date | DATE | Не из будущего |
| Текущий вес | input: number | DECIMAL (0.1 - 300.0) | Шаг 0.1 кг |
| Фотография | input: file | JPG / PNG | До 5 МБ |

### Запись на приём (раздел "Записаться")
| Поле | Источник | Формат |
| :--- | :--- | :--- |
| Питомец | select | ID питомца (выбор из своих) |
| Специализация | select | Терапевт, Хирург, Кардиолог, Невролог, Дерматолог |
| Врач | select | ID врача (фильтруется по специализации) |
| Дата приёма | input: date | DATE (>= текущей даты) |
| Время | select | Список слотов (09:00 - 15:00) |
| Комментарий | textarea | Текст (цель визита) |

---

## Выходная информация

### Основные документы и визуализация
| Документ | Получатель | Состав (выходные поля) |
| :--- | :--- | :--- |
| **Справка о посещении** | Владелец | Дата, Врач, Диагноз, Список назначений, Итоговая сумма |
| **Журнал здоровья** | Владелец | Хронология визитов, Анамнез, Рекомендации, Результаты анализов |
| **График веса** | Владелец | Отрисовка динамики массы (Canvas), дата измерения, вес в кг |
| **Аналитическая сводка** | Администратор | Выручка, Средний чек, Популярные услуги, Нагрузка врачей (%) |
