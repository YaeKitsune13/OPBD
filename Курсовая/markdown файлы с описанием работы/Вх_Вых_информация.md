# Описание входной и выходной информации

---

## Входная информация

### Регистрация владельца

| Поле | Источник | Формат |
| :--- | :--- | :--- |
| Фамилия, имя, отчество | Форма регистрации | Текст |
| Телефон | Форма регистрации | +7XXXXXXXXXX |
| Email | Форма регистрации | email |
| Пароль | Форма регистрации | Минимум 8 символов |

---

### Карточка питомца

| Поле | Источник | Формат |
| :--- | :--- | :--- |
| Кличка | Форма добавления питомца | Текст, только буквы |
| Вид и порода | Форма добавления питомца | Выбор из списка / текст |
| Дата рождения | Форма добавления питомца | DATE |
| Вес | Форма добавления питомца | DECIMAL, 0.1–300 кг |
| Фото | Форма добавления питомца | JPG / PNG, до 5 МБ |

---

### Запись на приём

| Поле | Источник | Формат |
| :--- | :--- | :--- |
| Питомец | Выбор из списка питомцев владельца | INTEGER (pet_id) |
| Врач | Выбор из списка врачей | INTEGER (doctor_id) |
| Дата и время | Выбор из свободных слотов | DATETIME |

---

### Медицинская карта визита (врач)

| Поле | Источник | Формат |
| :--- | :--- | :--- |
| Анамнез | Форма приёма | Текст, 10–2000 символов |
| Диагноз | Форма приёма | Текст, 3–500 символов |
| Услуги | Выбор из справочника | Список service_id + количество |
| Медикаменты | Выбор из справочника | Список medication_id + количество |
| Вес питомца | Форма приёма | DECIMAL, 0.1–300 кг |

---

## Выходная информация

| Документ | Получатель | Описание |
| :--- | :--- | :--- |
| Карточка визита | Владелец | Итог приёма: диагноз, назначения, стоимость |
| График веса | Владелец | Динамика веса питомца по датам |
| Расписание врача | Врач | Список записей на текущий день |
| История болезней | Врач | Все визиты выбранного питомца |
| Статус заявки | Владелец | Текущий статус записи на приём |

---

## Шаблоны выходных документов

           ВЕТЕРИНАРНАЯ КЛИНИКА
            Карточка визита №{visit_id}
    =======================================

    ПИТОМЕЦ
      Кличка:       {pet_nickname}

    АНАМНЕЗ
      {anamnesis}
    ---------------------------------------
    ДИАГНОЗ
      {diagnosis}

    НАЗНАЧЕНИЯ
      Услуги:
        - {service_name} x {qty}  —  {service_price} руб.
      Медикаменты:
        - {medication_name} x {qty}  —  {medication_price} руб.
    ---------------------------------------
    ИТОГО:          {total_cost} руб.
    =======================================

### Шаблон 2: Расписание врача на день

    =======================================
      РАСПИСАНИЕ НА {date}
      Врач: {doctor_last_name} {doctor_first_name}
    =======================================
     Время   | Питомец        | Владелец
    ---------|----------------|------------------
     {time}  | {pet_nickname} | {owner_full_name}
     {time}  | {pet_nickname} | {owner_full_name}
    =======================================
    Всего записей: {count}

---

### Шаблон 3: График веса питомца

    =======================================
      ДИНАМИКА ВЕСА: {pet_nickname}
      Владелец: {owner_full_name}
    =======================================
      Дата        | Вес (кг)
      ------------|----------
      {date}      | {weight}
      {date}      | {weight}
    =======================================
    Начальный вес: {first_weight} кг
    Текущий вес:   {last_weight} кг
    Изменение:     {delta} кг
