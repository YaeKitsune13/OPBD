# Определение сущностей и атрибутов

---

## Сущность: Владелец (Owner)

| Атрибут | Тип данных | Ограничения | Описание |
| :--- | :--- | :--- | :--- |
| **owner_id** | INTEGER | PK, NOT NULL, AUTO INCREMENT | Уникальный идентификатор |
| **last_name** | VARCHAR(50) | NOT NULL | Фамилия |
| **first_name** | VARCHAR(50) | NOT NULL | Имя |
| **middle_name** | VARCHAR(50) | NULL | Отчество |
| **address** | VARCHAR(255) | NULL | Домашний адрес |
| **phone** | VARCHAR(12) | NOT NULL, формат +7XXXXXXXXXX | Контактный телефон |
| **email** | VARCHAR(100) | NULL, формат email | Адрес электронной почты |
| **password_hash** | VARCHAR(255) | NOT NULL | Хэш пароля |

---

## Сущность: Питомец (Pet)

| Атрибут | Тип данных | Ограничения | Описание |
| :--- | :--- | :--- | :--- |
| **pet_id** | INTEGER | PK, NOT NULL, AUTO INCREMENT | Уникальный идентификатор |
| **owner_id** | INTEGER | FK → Owner, NOT NULL | Владелец питомца |
| **nickname** | VARCHAR(30) | NOT NULL, только буквы | Кличка |
| **species** | VARCHAR(50) | NOT NULL | Вид животного |
| **breed** | VARCHAR(50) | NULL | Порода |
| **birth_date** | DATE | NOT NULL | Дата рождения |
| **photo** | VARCHAR(255) | NULL, JPG или PNG | Путь к фотографии |
| **current_weight** | DECIMAL(5,2) | NOT NULL, 0.1–300 | Текущий вес (кг) |

---

## Сущность: История веса (WeightHistory)

| Атрибут | Тип данных | Ограничения | Описание |
| :--- | :--- | :--- | :--- |
| **weight_id** | INTEGER | PK, NOT NULL, AUTO INCREMENT | Уникальный идентификатор |
| **pet_id** | INTEGER | FK → Pet, NOT NULL | Питомец |
| **weight** | DECIMAL(5,2) | NOT NULL, 0.1–300 | Значение веса (кг) |
| **measured_at** | DATE | NOT NULL | Дата измерения |

---

## Сущность: Врач (Doctor)

| Атрибут | Тип данных | Ограничения | Описание |
| :--- | :--- | :--- | :--- |
| **doctor_id** | INTEGER | PK, NOT NULL, AUTO INCREMENT | Уникальный идентификатор |
| **last_name** | VARCHAR(50) | NOT NULL | Фамилия |
| **first_name** | VARCHAR(50) | NOT NULL | Имя |
| **middle_name** | VARCHAR(50) | NULL | Отчество |
| **specialization** | VARCHAR(100) | NOT NULL | Специализация |
| **phone** | VARCHAR(12) | NOT NULL | Контактный телефон |

---

## Сущность: Услуга (Service)

| Атрибут | Тип данных | Ограничения | Описание |
| :--- | :--- | :--- | :--- |
| **service_id** | INTEGER | PK, NOT NULL, AUTO INCREMENT | Уникальный идентификатор |
| **name** | VARCHAR(100) | NOT NULL | Наименование услуги |
| **description** | TEXT | NULL | Описание процедуры |
| **price** | DECIMAL(10,2) | NOT NULL, > 0 | Стоимость |

---

## Сущность: Медикамент (Medication)

| Атрибут | Тип данных | Ограничения | Описание |
| :--- | :--- | :--- | :--- |
| **medication_id** | INTEGER | PK, NOT NULL, AUTO INCREMENT | Уникальный идентификатор |
| **name** | VARCHAR(100) | NOT NULL | Название |
| **description** | TEXT | NULL | Описание |
| **price_per_unit** | DECIMAL(10,2) | NOT NULL, > 0 | Цена за единицу |

---

## Сущность: Запись на приём (Appointment)

| Атрибут | Тип данных | Ограничения | Описание |
| :--- | :--- | :--- | :--- |
| **appointment_id** | INTEGER | PK, NOT NULL, AUTO INCREMENT | Уникальный идентификатор |
| **pet_id** | INTEGER | FK → Pet, NOT NULL | Питомец |
| **doctor_id** | INTEGER | FK → Doctor, NOT NULL | Врач |
| **scheduled_at** | DATETIME | NOT NULL, > NOW() | Дата и время приёма |
| **status** | ENUM | NOT NULL | Ожидание / Подтверждено / Отклонено |

---

## Сущность: История статусов (AppointmentStatusLog)

| Атрибут | Тип данных | Ограничения | Описание |
| :--- | :--- | :--- | :--- |
| **log_id** | INTEGER | PK, NOT NULL, AUTO INCREMENT | Уникальный идентификатор |
| **appointment_id** | INTEGER | FK → Appointment, NOT NULL | Запись на приём |
| **status** | ENUM | NOT NULL | Новый статус |
| **changed_by** | VARCHAR(100) | NOT NULL | Автор изменения |
| **changed_at** | DATETIME | NOT NULL | Время изменения |

---

## Сущность: Медицинская карта визита (Visit)

| Атрибут | Тип данных | Ограничения | Описание |
| :--- | :--- | :--- | :--- |
| **visit_id** | INTEGER | PK, NOT NULL, AUTO INCREMENT | Уникальный идентификатор |
| **appointment_id** | INTEGER | FK → Appointment, NOT NULL, UNIQUE | Запись на приём |
| **anamnesis** | TEXT | NOT NULL, 10–2000 символов | Анамнез (жалобы и предыстория) |
| **diagnosis** | VARCHAR(500) | NOT NULL, 3–500 символов | Диагноз |
| **total_cost** | DECIMAL(10,2) | NOT NULL, вычисляется автоматически | Итоговая стоимость |

---

## Сущность: Назначение услуги (VisitService)

| Атрибут | Тип данных | Ограничения | Описание |
| :--- | :--- | :--- | :--- |
| **visit_service_id** | INTEGER | PK, NOT NULL, AUTO INCREMENT | Уникальный идентификатор |
| **visit_id** | INTEGER | FK → Visit, NOT NULL | Визит |
| **service_id** | INTEGER | FK → Service, NOT NULL | Услуга |
| **quantity** | INTEGER | NOT NULL, > 0 | Количество |

---

## Сущность: Назначение медикамента (VisitMedication)

| Атрибут | Тип данных | Ограничения | Описание |
| :--- | :--- | :--- | :--- |
| **visit_medication_id** | INTEGER | PK, NOT NULL, AUTO INCREMENT | Уникальный идентификатор |
| **visit_id** | INTEGER | FK → Visit, NOT NULL | Визит |
| **medication_id** | INTEGER | FK → Medication, NOT NULL | Медикамент |
| **quantity** | DECIMAL(6,2) | NOT NULL, > 0 | Количество (доз/упаковок) |
